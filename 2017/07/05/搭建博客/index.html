hexo 搭建博客(github)

Jul 5, 2017 | 杂学 | 阅读

步骤如下

1.新建仓库 username.github.io

2.新建read.me

3.安装node.js
4.安装git

5.安装hexo

6.修改hexo 的配置文件，和github仓库建立关联

7.购买域名，绑定username.github.io

8.发布博客 hexo clean &&hexo g &&hexo d

9.修改主题 详见

10.加入评论和订阅功能 详见此处

*遇到的问题：

1.解决npm 安装国外依赖速度比较慢的问题*

npm install cnpm -g --registry=https://registry.npm.taobao.org
安装的时候 用cpm 代替np
2.插入图片 直接拖进markdown编辑器就可以了，然后选择相应的图片就可以了。
